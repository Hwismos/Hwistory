Errors

FileNotFoundError

• 이 에러는 전달받은 경로상에 파일이 존재하지 않는 경우에 발생함

TypeError

• 이 에러는 함수 호출을 위해 전달된 데이터 유형이 잘못된 경우에 발생함

• [case 1]: TypeError: 'DGLREDataloader' object is not an iterator는 iterator 객체만을 인자로 받을 수 있는 next 함수에 iterator 객체가 아닌 DGLREDataloader 타입의 train_loader 객체가 전달되었기 때문에 발생함

• [case 2]: TypeError: cannot unpack non-iterable GCNLayer object는 self.GCNs이 GCN과 Attn로 분리될 수 없기 때문에 발생함

• [case 3]: TypeError: string indices must be integers는 문자열을 인덱싱하고자 할 때 정수가 아닌 값이 들어왔기 때문에 발생함

AttributeError

• 이 에러는 클래스 내에 정의되지 않은 속성이나 함수를 호출하려고 하는 경우에 발생함

• [case 1]: AttributeError: 'DGLREDataloader' object has no attribute 'next'는 next 함수를 정의하지 않은 DGLREDataloader 객체에서 next 함수를 호출했기 때문에 발생함

• [case 2]: AttributeError: 'str' object has no attribute 'read'는 str 객체에 read 함수가 정의되어 있지 않기 때문에 발생함

OSError

• 시스템 관련 에러. 'file not found error'와 'disk full' 등의 에러가 이 에러에 속함

• 'en_core_web_lg' 모델을 spacy를 통해 load 하려고 할 때에도 발생함. 이 경우에는 해당 wheel 파일을 다운 받은 뒤 python -m pip install ***.whl로 설치

• disk quota exceeded는 디스크 용량이 초과되어 발생함. du -m --max-depth=2 | sort -rn 커맨드로 파일들의 용량을 확인하고 불필요한 파일들을 제거해야 함. 모델 관련 수 많은 pt을 제거하였더니 83GB에서 4GB로 감소함

NameError

• 접근하고자 하는 변수나 함수가 선언되지 않은 경우에 발생함

IndexError

• 인덱스 범위를 넘어간 경우에 발생함

VScode의 SSH 에러

• ssh -i {key file} {User@HostName} 커맨드를 이용해 서버에 대한 접속 상태 확인. .vscode-server 폴더 삭제 후 vscode에서 다시 연결

SyntaxError와 RuntimeError

• SytaxError 파이썬 인터프리터가 찾아낸 에러 . RuntimeError는 CPU가 기계어로 변역된 프로그램을 수행하던 중 찾아낸 에러

UnboundLocalError

• 변수의 초기화가 조건문을 거치는 경우가 없는지 확인할 것
